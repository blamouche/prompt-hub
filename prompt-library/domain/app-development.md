# App development

- Maintenir un fichier de version à 3 chiffres `X.X.X` nommé `version.md`.
- Le chiffre `X.0.0` est créé uniquement à la demande de l'utilisateur.
- Le chiffre `0.X.0` est créé automatiquement à la création d'une nouvelle branche.
- Le chiffre `0.0.X` est incrémenté à chaque commit.
- Pour chaque action de développement de l'agent, il faut un commit.
